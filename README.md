# GitForDevops
Git For Devops

A beginner-friendly repository for learning and practicing Git commands commonly used in DevOps and software development workflows.

## 📚 About

This repository was created to practice Git fundamentals and build a personal reference guide for frequently used Git commands.

The objective is to understand:

- Git basics and version control
- Repository management
- Staging and committing changes
- Branching and merging
- Working with remote repositories
- Git workflows used in DevOps projects
